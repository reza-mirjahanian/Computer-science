## ğŸ’» **7. Algorithmic Identification (Python Example)**

You can programmatically find best responses and Nash equilibria:

```python
import numpy as np

# Payoff matrices: [Player1, Player2]
P1 = np.array([[5, 15],
               [10, 16]])

P2 = np.array([[6, 4],
               [2, 6]])

def find_nash(P1, P2):
    n_rows, n_cols = P1.shape
    nash = []
    for i in range(n_rows):
        for j in range(n_cols):
            # Is (i,j) a best response for P1?
            br1 = P1[i, j] == np.max(P1[:, j])
            # Is (i,j) a best response for P2?
            br2 = P2[i, j] == np.max(P2[i, :])
            if br1 and br2:
                nash.append((i, j))
    return nash

print("Nash Equilibria (row, col indices):", find_nash(P1, P2))
# Output: [(1, 1)] â†’ (Down, Right)
```

> ğŸ› ï¸ This brute-force method works for small finite games. For larger games, more advanced algorithms (e.g., Lemke-Howson) are used.

```python
        if br1[c] == r and br2[r] == c]
print("Nash equilibria (row, col):", nash)
# Output: [(1, 1)]
```
